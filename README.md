# My PersonalOS

# Description
This is a personal desktop-style OS.
Instead of making a simple website, I decided to make an operating system talking about myself.
It is an operating system where users can open draggable windows and open apps.
The site shows my interest, hobbies and ways to get in touch with me.

# Features
- Draggable windows
- Open/Close buttons
- Apps
- Time

# Tools
- HTML
- CSS
- JavaScript
- https://jams.hackclub.com/batch/webOS

# Challenges
- Making the windows draggable without overlapping
- Creating open/close apps 

# Learned
- How to add drag functions
- How to create window apps
